JAuthenticator is a Java based client for 2-factor authenticator used by Google.
Generating Google authentication codes, including HOTP generator specified by RFC 4226 and initially written by sweis@google.com (Steve Weis).
Can be used as a alternative to Google Authenticator mobile apps (iOS, Android).

How to run:
1. Checkout / download from GitLab.
2. cd to app folder
3. java -jar JAuthenticator.jar
4. Insert secret code
